2. Опишите способ определения и применения CSS-переменных (на примере).

Определение переменных : после (--) задаем имя, и затем значение, все это через селектор :root

:root { --maintextcolor: rgb(172, 164, 164); --unactivetextcolor: rgb(76, 71, 71); --fontmain: 20px; }

Вызываем при необходимости применить значение свойства к какому то типу элементов, с помощью: var(имя переменной). Можно применять многократно. Затем при замене знач. в самой переменной поменяется везде.

.title{ font-size: var(--fontmain); }

3. Опишите несколько семантических HTML-тегов и примеры их использования.

Семантические теги это как правило теги с говорящими именамии

header, main и footer как правило делят страницу на три смысловых блока. Например в хедере шапка с лого и меню, в мейн главная информация на странице и подвал для хранения контактных данных и т.д. Они могут использоваться не только для розделения всей страници, но и для смыслового разделения каких то блоков.

Есть так же менее глобальные но тоже говорящие теги - h1 h2 (заголовки)... nav (для навигационных ссылок) article и section (в остновном хранят главную информацию на сайте) aside (розмещается зачастую сбоку от основной информации)

4. Опишите разницу в значениях CSS-свойства position: static/absolute/fixed/sticky.

Static - положение блока по умолчанию, согласно положению в коде и CSS свойствам

Absolute - позиционирует елемент относительно границ екрана браузера с помощью дополнительных свойств top bottom left right. Если у одного из родительских блоков position reletive то будет позиционироватся внутри границ этого блока

Fixed - позиционирует елемент относительно границ екрана браузера с помощью дополнительных свойств top bottom left right. position reletive родительского блока на него никак не влияет

Sticky - Используется в основном при прокрутке, позиционируется внутри родительского блока, приклеивается к верху или низу окна браузера пока сам родительский блок на странице. Когла он исчезает то и сам блок Sticky исчезает вместе с ним. Позиционируется так же с помощью дополнительных свойств top bottom left right.
